# MongoDB Sorting & Limiting Commands

# Sorting Documents

### `db.<collection name>.find().sort({<fieldKey>: <sortType>})`

Sorts documents in ascending or descending order.

* `1` = ascending order
* `-1` = descending order

---

## Sort by Name

### Ascending Order (A → Z)

```
db.students.find().sort({name: 1})
```

### Descending Order (Z → A)

```
db.students.find().sort({name: -1})
```

---

## Sort by GPA

### Ascending Order (Lowest → Highest)

```
db.students.find().sort({gpa: 1})
```

### Descending Order (Highest → Lowest)

```
db.students.find().sort({gpa: -1})
```

---

# Limiting Documents

### `db.<collection name>.find().limit(<limit count>)`

Limits the number of displayed documents.

---

## Show Only 1 Document

```
db.students.find().limit(1)
```

---

## Show Only 3 Documents

```
db.students.find().limit(3)
```

---

# Combining Sort & Limit

### `db.<collection name>.find().sort({<fieldKey>: <sortType>}).limit(<limit count>)`

You can combine sorting and limiting in a single command.

---

## Highest GPA Student

```
db.students.find().sort({gpa: -1}).limit(1)
```

---

## Lowest GPA Student

```
db.students.find().sort({gpa: 1}).limit(1)
```

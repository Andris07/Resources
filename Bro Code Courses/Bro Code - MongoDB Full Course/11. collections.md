# MongoDB Collections

# Show Collections

### `show collections`

Displays all collections in the current database.

Example:

```
show collections
```

This command lists every collection inside the selected database.

---

# Create a Collection

### `db.createCollection("<collection name>")`

Creates a new collection.

Examples:

```
db.createCollection("teachers")
```

```
db.createCollection("courses")
```

This is the basic way to create collections in MongoDB.

---

# Create a Collection with Options

### `db.createCollection("<collection name>", {...})`

You can create collections with additional settings.

Example:

```
db.createCollection("teachers", {
    capped: true,
    size: 1024000,
    max: 100,
    autoIndexId: false
})
```

---

# Collection Options Explained

## `capped: true`

Creates a capped collection.

A capped collection:

* has a fixed size
* stores documents in insertion order
* automatically overwrites old documents when full

---

## `size`

Sets the maximum collection size in bytes.

Example:

```
size: 1024000
```

---

## `max`

Limits the maximum number of documents.

Example:

```
max: 100
```

This collection can store up to `100` documents.

---

## `autoIndexId`

Controls automatic `_id` index creation.

Example:

```
autoIndexId: false
```

In newer MongoDB versions, this option is mostly deprecated.

---

# Delete a Collection

### `db.<collection name>.drop()`

Deletes an entire collection permanently.

Example:

```
db.courses.drop()
```

This command deletes the `courses` collection and all of its documents.

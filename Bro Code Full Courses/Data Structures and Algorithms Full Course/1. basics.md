# Data Structures & Algorithms (D&A)

# What is a Data Structure?

A **data structure** is a way of organizing, storing, and managing data in a computer so that it can be accessed and modified efficiently.

Data structures define how data is arranged in memory and how operations such as searching, inserting, deleting, and updating can be performed.

---

## Examples of Data Structures

### Array

An **array** stores multiple values of the same type in a continuous block of memory.

Example:

```
[10, 20, 30, 40, 50]
```

Arrays allow fast access to elements using their index.

---

### Stack

A **stack** is a data structure that follows the **LIFO (Last In, First Out)** principle.

Example:

```
Push: 10, 20, 30

Stack:
30 ← Top
20
10
```

The last element added is the first element to be removed.

---

# What is an Algorithm?

An **algorithm** is a step-by-step set of instructions used to solve a problem or complete a task.

Algorithms take an input, process it through a sequence of steps, and produce an output.

---

## Main Parts of an Algorithm

An algorithm generally consists of:

* **Input** → The data provided to the algorithm.
* **Process** → The operations and steps performed on the data.
* **Output** → The final result generated by the algorithm.

---

## Why are Algorithms Important?

Algorithms help computers solve problems in an efficient and predictable way.

A good algorithm can:

* Reduce execution time.
* Use less memory.
* Handle larger amounts of data.
* Improve the performance of software.

---

## Example: Linear Search

A **linear search** is a simple algorithm used to find a specific value in a collection of data.

Example:

```
Array:
[4, 8, 15, 16, 23, 42]

Searching for: 16

Step 1: Check 4  → Not found
Step 2: Check 8  → Not found
Step 3: Check 15 → Not found
Step 4: Check 16 → Found
```

Linear search checks each element one by one until the target value is found or the end of the collection is reached.

---

# Why Do We Use Data Structures & Algorithms?

Data Structures and Algorithms (D&A) help programmers create software that is efficient, reliable, and scalable.

By choosing the right data structures and algorithms, developers can improve the speed, memory usage, and overall performance of their programs.

---

# Where Are D&A Used?

D&A are used in many areas of computer science, including:

* **Databases** → Organizing, storing, and searching large amounts of data.
* **Web Applications** → Managing users, content, and requests.
* **Operating Systems** → Handling memory management and task scheduling.
* **Artificial Intelligence** → Processing data and making decisions.
* **Computer Games and Graphics** → Managing objects, physics, and rendering calculations.

Understanding D&A is one of the most important skills for software developers because almost every software system relies on them.

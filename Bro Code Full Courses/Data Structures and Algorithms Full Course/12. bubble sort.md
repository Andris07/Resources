# Bubble Sort

# What is Bubble Sort?

**Bubble Sort** is a simple comparison-based sorting algorithm.

It repeatedly compares adjacent elements and swaps them if they are in the wrong order.

After each pass through the collection, the largest unsorted element "bubbles" toward the end of the array, which is where the algorithm gets its name.

Bubble Sort is commonly taught for educational purposes because it is easy to understand and implement.

---

# How Does Bubble Sort Work?

The algorithm repeatedly traverses the array and compares neighboring elements.

If two adjacent elements are in the wrong order, they are swapped.

Example:

```text
Initial Array

[9] [1] [8] [2]

Compare 9 and 1

[1] [9] [8] [2]

Compare 9 and 8

[1] [8] [9] [2]

Compare 9 and 2

[1] [8] [2] [9]
               ↑
Largest element moved to its final position
```

After every complete pass, one more element is guaranteed to be sorted.

# Visualization


The largest unsorted element moves toward the end during each pass.

---

# Pseudocode

```text
BubbleSort(array)

for i from 0 to length - 1

    for j from 0 to length - i - 1

        if array[j] > array[j + 1]

            swap elements
```

# Time Complexity

| Case         | Time Complexity |
| ------------ | --------------- |
| Best Case    | O(n²)           |
| Average Case | O(n²)           |
| Worst Case   | O(n²)           |

Bubble Sort compares many pairs of elements, making it inefficient for large datasets.

> Note: An optimized Bubble Sort implementation can achieve **O(n)** in the best case by detecting when no swaps occur during a pass.

---

# Space Complexity

```text
O(1)
```

Bubble Sort is an **in-place** sorting algorithm because it only uses a small number of additional variables.

---

# Advantages

Bubble Sort provides several advantages:

* Very easy to understand.
* Easy to implement.
* Uses constant extra memory.
* Good for educational purposes.
* Demonstrates the concept of sorting through swapping.

---

# Disadvantages

Bubble Sort also has several limitations:

* Very slow for large datasets.
* Performs many unnecessary comparisons.
* Inefficient compared to modern sorting algorithms.
* Rarely used in production environments.

---

# When Should You Use Bubble Sort?

Bubble Sort is useful when:

* Learning sorting algorithms.
* Teaching algorithm fundamentals.
* Working with very small datasets.
* Simplicity is more important than performance.

Typical examples include:

* Educational demonstrations.
* Algorithm visualization tools.
* Introductory programming courses.

---

# Why Is Bubble Sort Important?

Bubble Sort is often one of the first sorting algorithms taught in Data Structures and Algorithms.

Although it is inefficient for large datasets, it introduces important concepts such as:

* Comparison-based sorting.
* Swapping elements.
* Nested loops.
* Time complexity analysis.

Understanding Bubble Sort provides a foundation for learning more efficient sorting algorithms such as Insertion Sort, Merge Sort, Heap Sort, and Quick Sort.

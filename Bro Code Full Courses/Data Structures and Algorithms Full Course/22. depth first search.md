# Depth-First Search (DFS)

# What is Depth-First Search?

**Depth-First Search (DFS)** is a graph traversal algorithm that explores a graph by visiting one path as deeply as possible before backtracking.

Instead of visiting all neighboring vertices first, DFS continues moving forward until it reaches a vertex with no unvisited neighbors.

It then returns (backtracks) to the previous vertex and continues exploring the remaining paths.

---

# How Does DFS Work?

The algorithm follows these general steps:

1. Visit the starting vertex.
2. Mark it as visited.
3. Visit one unvisited neighboring vertex.
4. Repeat the process recursively.
5. When no unvisited neighbors remain, backtrack.
6. Continue until every reachable vertex has been visited.

Example:

```text
        A
      /   \
     B     C
    /     / \
   D     E   F
```

Possible DFS traversal:

```text
A → B → D → C → E → F
```

Unlike **Breadth-First Search (BFS)**, DFS always explores one branch completely before visiting another.

---

# Time Complexity

```text
O(V + E)
```

Where:

- **V** = Number of vertices
- **E** = Number of edges

Every vertex and edge is visited at most once.

---

# Space Complexity

```text
O(V)
```

The space complexity comes from:

- The recursion call stack.
- The visited collection.

---

# Advantages

DFS provides several advantages:

- Simple recursive implementation.
- Efficient graph traversal.
- Requires relatively little additional memory.
- Well suited for recursive problems.
- Forms the basis of many graph algorithms.

---

# Disadvantages

DFS also has several limitations:

- Does not guarantee the shortest path.
- Recursive implementations may cause stack overflow on very deep graphs.
- Can spend a long time exploring an unimportant branch before finding the desired vertex.

---

# DFS vs BFS

| Feature | DFS | BFS |
|---------|-----|-----|
| Data Structure | Stack / Recursion | Queue |
| Traversal Style | Deep First | Level by Level |
| Shortest Path | ❌ No | ✅ Yes (Unweighted Graphs) |
| Memory Usage | Usually Lower | Usually Higher |

---

# Where Is DFS Used?

Common applications include:

- Tree traversal.
- Maze solving.
- Cycle detection.
- Topological sorting.
- Connected component detection.
- Path finding.
- Backtracking algorithms.

---

# Why Is Depth-First Search Important?

Depth-First Search is one of the most fundamental graph traversal algorithms.

It introduces the concept of recursive graph exploration and serves as the foundation for many advanced algorithms used in computer science, including graph analysis, tree traversal, and numerous optimization techniques.

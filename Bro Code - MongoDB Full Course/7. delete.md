# MongoDB Delete Commands

# Delete One Document

### `db.<collection name>.deleteOne({<filter>})`

Deletes the first document that matches the filter.

Example:

```
db.students.deleteOne({name: "Larry"})
```

This command deletes the first student document where the name is `"Larry"`.

---

# Delete Multiple Documents

### `db.<collection name>.deleteMany({<filter>})`

Deletes all documents that match the filter.

---

## Delete Students Who Are Not Full-Time

```
db.students.deleteMany({fullTime: false})
```

This command deletes every student document where `fullTime` is `false`.

---

## Delete Documents Missing a Field

```
db.students.deleteMany({registerDate: {$exists: false}})
```

This command deletes all documents where the `registerDate` field does not exist.

---

# Important Notes

## `deleteOne()`

* Deletes only the first matching document.
* Stops after one deletion.

---

## `deleteMany()`

* Deletes all matching documents.
* Be careful when using broad filters.

---

## `$exists`

Checks whether a field exists in a document.

Example:

```
{registerDate: {$exists: false}}
```

* `$exists: true` → field exists
* `$exists: false` → field does not exist

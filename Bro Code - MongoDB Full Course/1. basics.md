# MongoDB & Shell Basics

## Shell Basic Commands

### `exit`

Closes the current terminal or shell session.

### `mongosh`

Starts the MongoDB Shell where you can work with databases and collections.

### `cls`

Clears the terminal screen.

---

# Basic MongoDB Commands

### `show dbs`

Displays all existing databases.

### `use <database name>`

Switches to an existing database.

Example:

```bash
use school
```

### `use <not existing database name>`

Creates and switches to a new database after data is added to it.

Example:

```bash
use newDatabase
```

### `db.createCollection("<collection name>")`

Creates a new collection inside the current database.

Example:

```javascript
db.createCollection("students")
```

### `db.dropDatabase()`

Deletes the currently selected database permanently.

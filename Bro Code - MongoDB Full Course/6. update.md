# MongoDB Update Commands

# Update One Document

### `db.<collection name>.updateOne({<filter>}, {<update>})`

Updates the first document that matches the filter.

---

## Update by Name

```
db.students.updateOne(
    {name: "Spongebob"},
    {$set: {fullTime: true}}
)
```

This command finds the student named `"Spongebob"` and sets the `fullTime` field to `true`.

---

## Update by ObjectId

```
db.students.updateOne(
    {_id: ObjectId('6a23e662149b644554abc114')},
    {$set: {fullTime: false}}
)
```

This command updates a document using its unique `_id` value.

---

## Remove a Field with `$unset`

```
db.students.updateOne(
    {name: "Gary"},
    {$unset: {fullTime: ""}}
)
```

The `$unset` operator removes a field completely from a document.

After running this command, the `fullTime` field will no longer exist in Gary's document.

---

# Update Multiple Documents

### `db.<collection name>.updateMany({<filter>}, {<update>})`

Updates all matching documents.

---

## Set `fullTime` to False for All Students

```
db.students.updateMany(
    {},
    {$set: {fullTime: false}}
)
```

An empty filter `{}` means every document in the collection will be updated.

---

## Add Missing Fields with `$exists`

```
db.students.updateMany(
    {fullTime: {$exists: false}},
    {$set: {fullTime: true}}
)
```

The `$exists` operator checks whether a field exists.

* `$exists: true` → field exists
* `$exists: false` → field does not exist

This command finds all students who do not have a `fullTime` field and adds it with the value `true`.

---

# Important Update Operators

## `$set`

Adds a new field or updates an existing field.

Example:

```
{$set: {fullTime: true}}
```

---

## `$unset`

Removes a field from a document.

Example:

```
{$unset: {fullTime: ""}}
```

---

## `$exists`

Checks if a field exists or not.

Example:

```
{fullTime: {$exists: false}}
```

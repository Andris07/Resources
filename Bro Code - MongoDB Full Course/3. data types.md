# MongoDB Data Types Example

This example shows different MongoDB data types inside a single document.

Example:

```
db.students.insertOne({
    name: "Larry",
    age: 32,
    gpa: 2.8,
    fullTime: false,
    registerDate: new Date("2023-01-02T00:00:00"),
    graduationDate: null,
    courses: ["Biology", "Chemistry", "Calculus"],
    address: {
        street: "123 Fake St.",
        city: "Bikini Bottom",
        zip: 12345
    }
})
```

---

# Data Types Explanation

### String

Text values are stored as strings.

Example:

```
name: "Larry"
```

---

### Number

Numbers can be integers or decimal values.

Examples:

```
age: 32
gpa: 2.8
```

---

### Boolean

Boolean values can only be `true` or `false`.

Example:

```
fullTime: false
```

---

### Date

Stores date and time values.

Example:

```
registerDate: new Date()
registerDate: new Date("2023-01-02")
registerDate: new Date("2023-01-02T00:00:00")
```

---

### Null

Represents an empty or unknown value.

Example:

```
graduationDate: null
```

---

### Array

Arrays store multiple values inside a list.

Example:

```
courses: ["Biology", "Chemistry", "Calculus"]
```

---

### Embedded Document (Object)

An object can store grouped related data.

Example:

```
address: {
    street: "123 Fake St.",
    city: "Bikini Bottom",
    zip: 12345
}
```

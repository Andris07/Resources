# MongoDB Find Commands

# Basic Find Method

### `db.<collection name>.find()`

Displays all documents inside a collection.

Example:

```
db.students.find()
```

---

# Find with Filters

### `db.<collection name>.find({<fieldKey>: <filterType>})`

Filters documents based on specific values.

---

## Find by Name

```
db.students.find({name: "Larry"})
```

---

## Find by GPA

```
db.students.find({gpa: 4.0})
```

---

## Find by Boolean Value

```
db.students.find({fullTime: false})
```

---

# Multiple Filters

### `db.<collection name>.find({<fieldKey>: <filterType>, <fieldKey>: <filterType>})`

You can combine multiple filters in one query.

Example:

```
db.students.find({gpa: 4.0, fullTime: true})
```

---

# Selecting Specific Fields

### `db.<collection name>.find({<fieldKey>: <filterType>}, {<fieldKey>: <boolValue>})`

Controls which fields are displayed.

* `true` = show field
* `false` = hide field

---

## Show Only the Name Field

```
db.students.find({}, {name: true})
```

---

## Hide the Name Field

```
db.students.find({}, {name: false})
```

---

## Hide `_id` and Show Only Name

```
db.students.find({}, {_id: false, name: true})
```

---

## Show Multiple Fields

```
db.students.find({}, {_id: false, name: true, gpa: true})
```

# MongoDB Comparison Operators

# Basic Syntax

### `db.<collection name>.find({<fieldKey>: {$<operator>: <fieldValue>}})`

Comparison operators are used to filter documents based on values.

---

# `$ne` — Not Equal

Finds documents where the value is not equal to the specified value.

Example:

```
db.students.find({name: {$ne: "Spongebob"}})
```

This command finds all students except `"Spongebob"`.

---

# `$lt` — Less Than

Finds values smaller than the specified value.

Example:

```
db.students.find({age: {$lt: 20}})
```

This command finds students younger than `20`.

---

# `$lte` — Less Than or Equal

Finds values smaller than or equal to the specified value.

Example:

```
db.students.find({age: {$lte: 27}})
```

This command finds students who are `27` years old or younger.

---

# `$gt` — Greater Than

Finds values greater than the specified value.

Example:

```
db.students.find({age: {$gt: 27}})
```

This command finds students older than `27`.

---

# `$gte` — Greater Than or Equal

Finds values greater than or equal to the specified value.

Example:

```
db.students.find({age: {$gte: 27}})
```

This command finds students who are `27` years old or older.

---

# Combining Comparison Operators

You can combine multiple comparison operators in the same query.

Example:

```
db.students.find({gpa: {$gte: 3, $lte: 4}})
```

This command finds students with a GPA between `3` and `4`.

---

# `$in` — Match Any Value in a List

Finds documents where the value matches any item in an array.

Example:

```
db.students.find({
    name: {$in: ["Spongebob", "Patrick", "Sandy"]}
})
```

This command finds students whose names are:

* `"Spongebob"`
* `"Patrick"`
* `"Sandy"`

---

# `$nin` — Not in List

Finds documents where the value is not in the array.

Example:

```
db.students.find({
    name: {$nin: ["Spongebob", "Patrick", "Sandy"]}
})
```

This command finds all students except:

* `"Spongebob"`
* `"Patrick"`
* `"Sandy"`
